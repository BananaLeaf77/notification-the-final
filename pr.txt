Pada halaman ini user admin dan guru / staff dapat melakukan autentikasi untuk akses kedalam sistem,
setelah login berhasil sistem akan memecah alur ke dashboard menjadi 2 yaitu untuk admin dan untuk guru.
Pada halaman ini juga terdapat fungsi mengirim permintaan perubahan data.

-2. dashboard admin
halaman ini adalah halaman yang akan dialihkan setelah melakukan autentikasi pada halaman login . 
pada halaman ini hanya user dengan role admin yang dapat mengakesnya. Halaman dashboard admin ini terdapat fungsi pengelolaan siswa, orang tua dan notifikasi absensi.

-3. dashboard guru / staff
halaman ini adalah halaman yang akan dialihkan setelah melakukan autentikasi pada halaman login . 
pada halaman ini hanya user dengan role guru / staff yang dapat mengakesnya. Halaman dashboard guru / staff ini terdapat fungsi pengelolaan notifikasi absensi.

-4. test score input admin 
pada halaman ini hanya user dengan role admin yang dapat mengaksesnya. halaman ini digunakan untuk penginputan masing masing nilai siswa dan pengiriman pengumuman nilai tes

-5. test score input guru / staff 
pada halaman ini hanya user dengan role guru / staff yang dapat mengaksesnya. halaman ini digunakan oleh guru / staff untuk melakukan penginputan nilai masing masing siswa yang diajarnya saja.

-6. registered teacher / guru terdaftar
pada halaman ini hanya user dengan role admin yang dapat mengaksesnya. halaman ini berguna untuk user admin melakukan pengelolaan guru / staff

-6. Halaman detail guru / staff
Halaman ini dapat diakses oleh admin dan guru / staff. halaman ini menampilkan data detail dari suatu guru / staff.

-8. Halaman detail siswa dan orang tua
halaman ini dapat diakses oleh admin dan guru / staff. halaman ini berisi data detail dari siswa dan orang tua yang tersangkut.

9. Halaman Unggah CSV
